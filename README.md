# vue-study-todo

Vue.js勉強会において使用するレポジトリです。

# 環境構築

## レポジトリをクローン

```
git clone https://github.com/azukiazusa1/vue-study-todo.git
```

## フロントエンド

```
cd frontend
npm install
npm run serve
```

## バックエンド

```
cd backend
npm install
npm run serve
```

各ステップごとにブランチが別れています。
途中で詰まったら、ブランチを切り替えて進めてください。
